Comparison and Evaluation

After using both Fireworks and Dreamweaver to create a home page, I have noticed many differences between them.
Fireworks is more user friendly because you just need to create the images instead of typing html/css. You don't need to
know html/css to make your homepage on fireworks. Also it takes less time to create your home page on Fireworks than Dreamweaver.
However using Dreamweaver allows you to have more customization and it is easier to change something. If you want to change the background
color all you need to do is change the hex code for the color. Also since the Fireworks version is all images you
can't search them on google but you can for Dreamweaver.

    Fireworks
  • More user friendly
  • Don't need to know html/css
  • Quicker to create a homepage
  • Less customization
  • Harder to change something
  • Can't search the text on google
    Dreamweaver
  • Not as user friendly
  • Need to know how to use html/css
  • Takes longer to create a homepage
  • More customization
  • Easier to change something
  • Can search the text on google
