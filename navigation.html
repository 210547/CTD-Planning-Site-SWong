Graphical Navigation

To add graphical navigation into my planning site, I used Fireworks and Dreamweaver. I created the buttons in Fireworks, and added a different look for unpressed, hovered, and pressed. Then, I exported it into html. In Dreamweaver, I imported the html file and then put it into my planning site template. Since there was a bug with the importing, the buttons did not change when interacted with. I had to copy the script over from the html file I exported from Fireworks into the <head> of the template. I also had to relink all of the images because the pictures were linked incorrectly.
